# 💻 Real-Time Todo Management System

A professional Todo Management System developed using **Node.js**, **Express.js**, **EJS**, and **Bootstrap 5**.

This project allows users to:

- Add Tasks
- Edit Tasks
- Delete Tasks
- Manage Task Status
- View Dashboard Statistics

---

# 🚀 Technologies Used

- Node.js
- Express.js
- EJS
- Bootstrap 5
- CSS3

# ✨ Features

## 📊 Dashboard Module

- Display total tasks
- Display pending tasks
- Display completed tasks
- Responsive dashboard cards

---

## ➕ Add Task Module

Users can:

- Add Task ID
- Add Task Name
- Add Course Name
- Select Task Status

After submission, the task is added instantly to the dashboard.

---

## 📝 Edit Task Module

Users can:

- Edit existing task details
- Update task status
- View pre-filled form data

---

## ❌ Delete Task Module

Users can:

- Delete tasks permanently
- Confirmation popup before deletion

---

## 🔄 Status Management

Task statuses include:

- Pending
- In Progress
- Completed

Status badges change dynamically.

# ⚙️ Installation Steps

## 1️⃣ Clone Project

```bash
git clone <repository-url>
```

---

## 2️⃣ Open Project Folder

```bash
cd TodoApp
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Server

```bash
node app.js
```

---

# 🌐 Run In Browser

```bash
http://localhost:8000
```
